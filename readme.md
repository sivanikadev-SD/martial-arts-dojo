# Martial Arts Dojo Dashboard Template

A premium, modern, and responsive template designed for Martial Arts schools, Dojos, and combat sports academies.

## 🥋 Key Features
- **Dual Homepages**: Choose between a high-energy action layout (Home 1) or a modern tech-style split layout (Home 2).
- **Comprehensive Dashboards**: Feature-rich student portal with belt tracking and an admin management interface.
- **Dark/Light Mode**: Seamless theme switching with system detection.
- **RTL Support**: Full compatibility for Arabic and Hebrew layouts.
- **Responsive & Accessible**: Mobile-first design adhering to WCAG 2.1 AA guidelines.
- **Pre-built Components**: Skeleton loaders, interactive progress bars, pricing cards, and more.


## 🚀 Quick Start
1. Clone or download the template.
2. Open `pages/index.html` in any modern browser.
3. To customize colors, edit the CSS variables in `assets/css/style.css`.

## 🛠 Tech Stack
- **HTML5**: Semantic markup.
- **CSS3**: BEM methodology, Modern CSS Variables.
- **JavaScript**: ES6+ Modular logic.
- **Typography**: Outfit (Headings) & Inter (Body).
